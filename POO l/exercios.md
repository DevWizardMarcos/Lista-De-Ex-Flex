
**Desafio 1: Cadastro de Pessoas**
Implemente um sistema para cadastro de pessoas. Crie uma classe chamada `Pessoa` que represente um indivíduo no sistema. Em seguida, instancie uma pessoa e exiba seus dados, simulando o registro de um novo usuário.

---

**Desafio 2: Gerenciamento de Frota**
Você foi contratado para desenvolver parte de um sistema de gerenciamento de frota de veículos para uma empresa. Crie uma classe `Carro` com os atributos: marca, modelo e ano. Simule o cadastro de um carro e exiba suas informações, como seria feito em um painel administrativo.

---

**Desafio 3: Sistema Acadêmico**
Desenvolva um módulo para um sistema acadêmico. Crie uma classe `Aluno` com os atributos: nome, idade e curso. Cadastre dois alunos diferentes e mostre os dados de cada um, como se estivesse exibindo o perfil dos estudantes em um portal universitário.

---

**Desafio 4: Pet Shop Virtual**
Implemente parte de um sistema para um pet shop virtual. Crie uma classe `Cachorro` com um atributo `nome` e um método `latir()` que exiba uma mensagem personalizada com o nome do cachorro, simulando uma interação no site.

---

**Desafio 5: Banco Digital**
Você está desenvolvendo um protótipo de banco digital. Crie uma classe `ContaBancaria` com um atributo `saldo` e métodos para depositar e sacar valores. Simule operações de depósito e saque, mostrando o saldo atualizado após cada operação.

---
